# 🚀 Space Escape Runner

A fun 2D arcade space game built using **React Native** and **Expo** where players control a spaceship and dodge incoming asteroids while trying to achieve the highest score.

---



## ✨ Features

- 🚀 Modern spaceship design
- ☄️ Animated rotating asteroid
- 🌌 Animated star background
- 📈 Dynamic difficulty (asteroid speed increases)
- 🏆 High Score using AsyncStorage
- 📳 Haptic feedback
- 🔄 Restart Game
- 🎨 Modern UI
- 📱 Android APK support
- ⚡ Built using Expo

---

## 🛠 Tech Stack

- React Native
- Expo
- TypeScript
- AsyncStorage
- Expo Haptics
- Expo Linear Gradient
- Expo EAS Build

## ▶️ Installation

Clone the repository

```bash
git clone https://github.com/adithya-adari/Space-Escape-Runner.git
```

Install dependencies

```bash
npm install
```

Run the project

```bash
npx expo start
```

---

## 📦 Build APK

```bash
eas build --platform android --profile preview
```
